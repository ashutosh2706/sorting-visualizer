# Sorting Visualizer

*A software tool to visualize various sorting algorithms.*

- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort
- Heap Sort

<br>

## 👨‍💻 Usage :
- Select the algorithm you want to visualize from the dropdown menu.
- Press the "START" button to see the animation of the algorithm in action.
